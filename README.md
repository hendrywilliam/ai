# AI

https://ai.hendryw.com/

A simple AI powered application, Build with Next.js.

Key features: 

- [x] Text Generation/Chat Completions

# Setup
* Run `pnpm install`
* Add .env file
  
  ```
  NODE_ENV=
  OPENAI_API_KEY=
  ```
* Run `pnpm run dev` for development build / `pnpm run build` then `pnpm run start` for production build.
